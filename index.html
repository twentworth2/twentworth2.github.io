    FindTune

    The goal of this project was to build an audio harmonizer using the DE1-SoC FPGA board by Altera. We wanted to use either the board’s line in or microphone's line in to play a note, voice, or other music to the board. We then would process the audio signal and output the harmonized note through the board’s line out.

      Was chosen as leader of group of six.

      Created a custom compiler, assembler, and ISA.

      Designed and created a 5 stage processor with pipelining, forwarding, and branch prediction.

      Group was able to finish project but unable to test on hardware due to COVID-19.

    Bop It!

    In this project a partner and I programmed a TIVA launchpad using C++ to play a version of the childrens game Bop It! The goal was to design a project that showcased all of the functions of our board.

      Initialized and used the joystick, pushbuttons, timers, LEDs, touchpad, LCD, accelerometer and switches. Utilized interrupts and interrupt handling for the pushbuttons. Project also included the use of a UART and a EEPROM.

      Throughout the semester we edited and tested source files to learn what they were and how they worked.

      Not all source code is ours, some was provided by the class.

      Code written by my team can be found in the project folder on github.
